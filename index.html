公司简介

瑞途科技有限公司（RuiTu Technology）成立于 2012 年 4 月，总部位于北京中关村科技园区。作为国内领先的智能手机创新企业，我们始终专注于新一代智能终端研发与制造，累计服务全球超 5000 万用户。经过 13 年高速发展，公司现有员工 1200 余人，构建了涵盖研发中心、智能制造基地、全球营销中心、供应链管理部及用户体验研究院五大核心部门的完整产业体系。

招聘职位

软件工程师

职位描述：负责公司软件产品的开发、测试和维护工作，参与项目的需求分析和设计。

任职要求：熟悉 Java 或 Python 编程语言，有良好的编程习惯和团队协作能力。

数据分析师

职位描述：收集、整理和分析业务数据，为公司决策提供数据支持，撰写数据分析报告。

任职要求：熟练掌握 SQL 语言，具备数据分析工具（如 Excel、Python 数据分析库）的使用经验。
